كيفية الربط مع Google Sheets
1. افتح Google Cloud Console وأنشئ مشروعاً جديداً.
2. فعّل Google Sheets API وGoogle Drive API.
3. أنشئ Service Account وحمّل ملف JSON.
4. شارك جدول البيانات مع البريد الإلكتروني للـ Service Account.
5. انسخ Spreadsheet ID من رابط الجدول.
من رابط جدول البيانات: docs.google.com/spreadsheets/d/ID/edit
